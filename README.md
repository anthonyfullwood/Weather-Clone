# Weather-Clone
This project is a based on the default Apple Weather App.<br />

The technology stack used in this procject: <br />

  - Rest API
  - Core Location
  - UIKit
  
Features: <br />

  - Get current weather of city by searching its name
  - Get current weather for your location
